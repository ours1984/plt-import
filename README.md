## 背景

PLT文件源自于HP绘图仪，PLT文件格式是基于矢量的，由Hewlett Packard开发。在诸如AutoCAD这样的程序中，用于打印绘图仪上的绘图。其它应用程序，例如CorelDraw ，可以解释HPGL的子集以及HPGL/2 命令集。
